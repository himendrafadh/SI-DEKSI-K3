# SI-DEKSI K3 Industrial Safety Dashboard

## Navigation Guide

### Main Dashboard (Monitoring System)
Access the main monitoring dashboard with sidebar navigation:
- **URL**: `/` (home)
- **Features**: Real-time monitoring, Logs, Analytics
- **Includes**: Navigation sidebar, Status bar, System indicators

### Check-In Kiosk View
Access the dedicated check-in terminal for gate entrance:
- **URL**: `/check-in`
- **Features**: 
  - Full-screen kiosk interface
  - Live video feed with YOLO bounding boxes
  - HOG face recognition
  - Real-time PPE detection checklist
  - Visual guidance for missing equipment
  - Access grant/deny status banner
- **Optimized for**: 24-inch kiosk displays (vertical or horizontal)
- **Use case**: Worker check-in at entrance gates

### Quick Access
From the main dashboard, click the **"Open Check-In Kiosk"** button in the status bar to open the kiosk view in a new tab.
